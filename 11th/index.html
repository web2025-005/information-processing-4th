他社との違い

ユニクロの商品と他社の商品でどのような違いがあるのかポイントをまとめました。

他社との違い
値段 h1, h2, p, ul, table などのタグで、Webページの骨組み（構造）を作る
品質 h1 はページ全体の大見出し、h2 はセクションの見出しとして使い分けることで、読者が内容を理解しやすくなる
店舗数 外部ファイル（style.css）で見出しの色や表のデザインを一括に変更でき、複数ページで統一できる
SDGs ページ間をリンク（a タグ）で結ぶことで、複数ページが1つのサイトになる
客層 img タグに alt 属性をつけることで、画像が表示できない場合でも情報が伝わる
トレンド性 参考にしたページのURLと参照日を正確に記載することで、情報の信ぴょう性が高まる
